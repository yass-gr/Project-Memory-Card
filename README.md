# Memory Card Game (unfinished)

A memory card matching game built with React, Tailwind CSS, and GSAP animations.

## About the Game

Don't click the same card twice! Cards are shuffled after every click, making it challenging to remember which ones you've already revealed. Click each unique card exactly once to win - click a card you've already clicked and you lose.

## Preview

https://project-memory-card-gamma-woad.vercel.app/

## Tech Stack

- **React** - Learning React fundamentals
- **Tailwind CSS** - Styling with utility-first CSS
- **GSAP** - Animation library (first time learning)
- **Vite** - Build tool

## Getting Started

